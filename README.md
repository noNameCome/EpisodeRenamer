# RenaMatcher

드래그앤드롭으로 영상/자막 파일명을 한번에 정리해주는 프로그램입니다.

---

## 이런 분께 추천해요

- 애니메이션이나 드라마를 다운받았는데 영상이랑 자막 파일명이 달라서 자막이 안 붙는 분
- 파일명을 `레일 워즈 - 01화.mp4`, `레일 워즈 - 02화.mp4` 처럼 깔끔하게 정리하고 싶은 분
- 파일명 하나하나 손으로 바꾸기 귀찮은 분

---

## 실행 방법

### 방법 1 - 그냥 실행 (추천)
[Releases](../../releases) 에서 `RenaMatcher.exe` 다운받아서 바로 실행하세요.  
Python 설치 없이 바로 사용할 수 있습니다.

### 방법 2 - 소스코드로 실행
Python 3이 설치되어 있어야 합니다.

```bash
pip install pillow tkinterdnd2
python main.py
```

---

## 사용 방법

### 1. 폴더 열기
- 정리할 파일이 있는 폴더를 프로그램 창에 **드래그앤드롭**
- 또는 폴더에서 **우클릭 → "RenaMatcher로 열기"** (레지스트리 등록 후 사용 가능)

### 2. 변경 모드 선택

| 모드 | 설명 |
|------|------|
| 비디오 기준 | 영상 파일명에 맞춰 자막 파일명을 변경 |
| 자막 기준 | 자막 파일명에 맞춰 영상 파일명을 변경 |
| 이름규칙 | 직접 규칙을 입력해서 번호 순서대로 변경 |

### 3. 이름규칙 사용법 (이름규칙 모드)

| 입력 | 결과 예시 |
|------|-----------|
| `레일 워즈 %d화` | 레일 워즈 1화, 레일 워즈 2화 ... |
| `레일 워즈 %02d화` | 레일 워즈 01화, 레일 워즈 02화 ... |
| `레일 워즈 - EP%02d` | 레일 워즈 - EP01, 레일 워즈 - EP02 ... |

### 4. 미리보기 확인 후 적용
- **일괄수정** 버튼을 누르면 변경 전/후 미리보기가 뜹니다
- 확인하고 적용하면 파일명이 바뀝니다

### 5. 실수했을 때
- **복원** 버튼으로 원래 파일명으로 되돌릴 수 있습니다

---

## 우클릭 메뉴 등록

폴더에서 우클릭 했을 때 "RenaMatcher로 열기" 메뉴를 추가할 수 있습니다.

- **레지스트리 등록** 버튼 클릭 → 우클릭 메뉴에 추가됨
- **레지스트리 제거** 버튼 클릭 → 우클릭 메뉴에서 제거됨
